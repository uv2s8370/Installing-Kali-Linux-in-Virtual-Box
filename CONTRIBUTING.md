# Contributing to Guide  
Thank you for being part of this project! To keep things organized, please follow these guidelines.  

**How You can contribute**  
- Fix typos or grammar
- Improve explanations
- Add screenshots or examples
- Update steps if anything changes
- Suggest better formatting

🍴 **Getting Started**
- Fork the repository
- Clone your fork to your local machine:
  git clone https://github.com/uv2s8370/Installing-Kali-Linux-in-Virtual-Box.git
- Create a new branch for your changes:
  git checkout -b Improved-Guide
- Make your changes
- Commit your changes:
  git commit -m "Improved Guide steps"
- Push to your fork:
  git push origin Improve-Guide
- Open a Pull Request

# Guidelines
- Keep instructions simple and beginner-friendly
- Use proper Markdown formatting
- Test your changes before submitting
- Write clear commit messages

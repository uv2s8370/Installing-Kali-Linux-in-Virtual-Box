# Installing-Kali-Linux-in-Virtual-Box
Kali Linux provides multiple options to install it in computer. I am going to talk about VM installation.
I am using Virtual Box from Oracle as my VM. 

# 🛠️**Requirements**
* Virtual Box
* Kali Linux VM image
* WinRAR/7zip 
* Minimum 8 GB RAM
* ~30 GB storage

**Step1-**  
- Download Virtual Box from official Oracle website  
- Run .exe file just downloaded  
- Follow on screen instructions to install the Virtual Box  

**Step2-**  
- Download Kali Linux VM image from official website of Kali  
- The file will be in .7z format  
- Extract the file on your hard disk using WinRar or 7zip  
- Upon extraction you will find two files. We will need file with the extension ".vbox"  

**Step3-**  
- Run Virtual Box  
- On Home Screen click "Machine" option which is located at top left hand corner   
- From dropdown select "Open" option   
- It will open a small window  
- Locate the .vbox file that we just unzipped and click "open"  
- It will install the Kali Linux in VM with default settings  
- You can change the settings later on by going to "setting" option  

  ⏯️ **Usage**
  Login with:
  - Username: kali
  - Password: kali
